ENDLESS WAR, YOHA WITH MATTHEW FULLER (2011 - 2013, video based installation)

Endless war is a software driven system that processes the data set obtained from the WikiLeaks Afghan War Diary as a collection of machine and human analytic viewpoints. The installation consist of a computer processing data in real time, a projector displaying results and a coil pick up microphone that sonifies the inner working of the machine. The project critically reflects on computers and database machines recording military acts.
